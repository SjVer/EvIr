\doxysubsection{Detailed Description}
A single basic block containing instructions. 

A basic block is simply a container of instructions that execute sequentially. Proper basic blocks should have a list of non-\/terminating instructions followed by a single terminator instruction. Terminator instructions may not occur in the middle of a basic block, and must terminate the block. This is not enforced during the \char`\"{}construction\char`\"{} of the IR.
